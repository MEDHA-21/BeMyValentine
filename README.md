A romantic Valentine's Day website for your special someone! Built with React and ready for Netlify deployment.

## Features

- **Landing Page**: A beautiful "Will you be my Valentine?" question with interactive buttons
- **13 Reasons Why I Love You**: Click-to-reveal cards with personalized love messages
- **Photo Memory Quiz**: Test how well your partner remembers your special moments
- **Floating Hearts**: Romantic animated hearts floating across the screen
- **Background Music**: Add your favorite love song for an immersive experience
- **Confetti Celebration**: Celebration effects when they say YES!

## Getting Started

### Prerequisites
- Node.js 18+ installed

### Installation

```bash
# Install dependencies
npm install

# Start development server
npm run dev
```

### Customization

#### 1. Add Your Photos
Place your photos in the `public/images/` folder:
- `img1.jpg`
- `img2.jpg`
- `img3.jpg`
- `img4.jpg`
- `img5.jpg`

#### 2. Add Background Music
Place your love song as `public/music/love-song.mp3`

#### 3. Customize the 13 Reasons
Edit `src/pages/Reasons.jsx` to add your own personal reasons

#### 4. Customize Quiz Questions
Edit `src/pages/Quiz.jsx` to match your photos with the correct dates and memories

#### 5. Personalize Messages
Edit `src/pages/Home.jsx` to customize the love message
